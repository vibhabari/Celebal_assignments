#  Spark Basics Assignment

## Objective
Learn the basics of Apache Spark and use DataFrames for data cleaning, transformation, and analysis.

## Steps Performed
1. Learn Basics (MapReduce, Spark, Dataframes) 
2. Created Spark Session.
3. Loaded CSV data into DataFrame.
4. Removed duplicate records.
5. Handled missing values.
6. Applied filters on age, category, and region.
7. Renamed columns and changed data types.
8. Performed aggregations:
   - Count
   - Average
   - Minimum
   - Maximum
9. Used groupBy() for regional analysis.
10. Built a complete Spark pipeline.
11. Saved results to output folder.

## Observations

- Spark processed data efficiently using DataFrames.
- Duplicate and null records were removed successfully.
- GroupBy operations generated summarized insights.
- Aggregation functions helped analyze sales trends.
- Spark is faster than MapReduce because it performs computations in memory.
