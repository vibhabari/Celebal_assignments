# Sales Data Analysis using SQL 
* Use Subqueries, CTEs, and Window Functions to analyze sales data from the Superstore dataset.

## Task 1: Load Dataset

* Loaded the Superstore dataset into the `superstore_raw` table.
* This table was used for all further analysis.

---

## Task 2: Create Tables

* Created `customers`, `orders`, and `products` tables using `SELECT DISTINCT`.
* This helped organize the data properly.

---

## Task 3: Insert Data

* Inserted unique records into all tables.
* Removed duplicate data for better analysis.

---

## Task 4: Use Subqueries

* Found orders with sales greater than average sales.
* Identified the highest order for each customer.
* Helped identify important customers and high-value orders.

---

## Task 5: Use CTEs

* Calculated total sales for each customer using CTEs.
* Made complex queries simpler and easier to read.

---

## Task 6: Use Window Functions

* Used `ROW_NUMBER()` and `RANK()` to rank customers by sales.
* Helped find top-performing customers.

---

## Task 7: Combine JOIN + CTE + Window Functions

* Combined multiple SQL concepts to create final customer sales reports.
* Generated complete customer performance analysis.

---

## Task 8: Solve Business Queries

### Top Customers

* Identified customers with highest sales.

### Low Customers

* Found customers with low sales.

### Single-Order Customers

* Found customers who ordered only once.

### Above-Average Sales

* Identified customers with sales above average.

---

# Conclusion

This project helped in understanding advanced SQL concepts like subqueries, CTEs, joins, and window functions. The analysis provided useful business insights from sales data.
